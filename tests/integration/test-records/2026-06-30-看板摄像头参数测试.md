## 详细测试记录

### TC1-3: 统计总览接口

```
# 鉴权
GET /api/statistics/overview (无token) → 401 ✅

# 正常请求
GET /api/statistics/overview (有效token)
→ 200 {
  "today_event_count": N,
  "total_event_count": N,
  "recent_events": [...],
  "daily_trend": [...]
}
字段类型: today_event_count(int), total_event_count(int), recent_events(array, ≤5), daily_trend(array, 7天)
```

### TC4-8: 摄像头MJPEG流

```
# 启动
POST /api/camera/start {camera_index:0, width:640, height:480}
→ 200 {"status":"started","camera_index":0}

# 状态
GET /api/camera/status
→ 200 {"status":"online","camera_index":0}

# MJPEG流
GET /api/camera/stream (无鉴权)
→ 200 Content-Type: multipart/x-mixed-replace; boundary=frame
帧格式: --frame\r\nContent-Type: image/jpeg\r\n\r\n<JPEG bytes>

# 重复启动
POST /api/camera/start {camera_index:0}
→ 200 {"status":"already_running"}

# 停止
POST /api/camera/stop
→ 200 {"status":"stopped"}

# 停止后状态
GET /api/camera/status
→ 200 {"status":"offline"}
```

### TC9-11: 参数读写

```
# 读取
GET /api/settings
→ 200 {detection_threshold, tracking_iou_threshold, ...6个参数}

# 更新
PUT /api/settings {detection_threshold: 0.45}
→ 200

# 验证持久化
GET /api/settings
→ detection_threshold = 0.45 ✅

# 无效值
PUT /api/settings {detection_threshold: "abc"}
→ 400 ✅
```
