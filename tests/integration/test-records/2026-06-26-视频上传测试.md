## 1. 视频上传 (6 tests)

### API-UP-001: 正常MP4上传
```bash
curl -X POST http://127.0.0.1:8000/api/videos/upload \
  -H "Authorization: Bearer $TOKEN" \
  -F "file=@test.mp4"
```
**预期**: code=200, 返回 task_id, filename, total_frames
**结果**: ✅ PASS — 返回 `{"code":200, "data":{"task_id":1, "filename":"xxx_test.mp4","size":...,"total_frames":240}}`
