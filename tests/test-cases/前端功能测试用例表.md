# 前端功能测试用例表

> 基于 [刘康个人开发计划.md](../../刘康个人开发计划.md) 第九节边缘情况清单扩展  
> 覆盖8个页面的功能、交互、边界和异常场景

---

## 1. 登录页 `/login`

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|---|---|---|---|
| FE-LOGIN-001 | 正确登录 | 输入admin/admin123，点击登录 | 跳转系统首页`/`，localStorage存有token |
| FE-LOGIN-002 | 错误密码 | 输入admin/wrong，点击登录 | 表单下方显示"账号或密码错误"，不跳转 |
| FE-LOGIN-003 | 空用户名 | 用户名为空，输入密码，点击登录 | 表单校验提示"请输入账号" |
| FE-LOGIN-004 | 空密码 | 输入用户名，密码为空，点击登录 | 表单校验提示"请输入密码" |
| FE-LOGIN-005 | 已登录访问登录页 | 已有有效token，访问`/login` | 自动跳转系统首页`/` |
| FE-LOGIN-006 | 后端离线时登录 | 后端未启动，输入正确账号密码，点击登录 | 显示"无法连接服务器"或网络错误提示 |

## 2. 系统首页 `/`

## 3. 视频分析页 `/analysis`

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|---|---|---|---|
| FE-ANALYSIS-001 | 选择视频文件 | 点击选择文件，选一个MP4 | 显示文件名 |
| FE-ANALYSIS-002 | 上传正常视频 | 选择MP4后自动上传（或点击上传） | 任务创建成功，显示task_id |
| FE-ANALYSIS-003 | 上传非视频文件 | 选择.txt文件并上传 | 显示错误提示"仅支持mp4、avi、mov、mkv" |
| FE-ANALYSIS-004 | 上传超大文件 | 选择超过限制大小的视频 | 显示"文件过大"提示 |
| FE-ANALYSIS-005 | 视频首帧显示 | 上传成功后 | 显示视频首帧图片，Canvas区域可见 |
| FE-ANALYSIS-006 | ROI矩形绘制 | 鼠标在首帧上拖拽 | 显示矩形框，可拖拽调整位置和大小 |
| FE-ANALYSIS-007 | ROI坐标传递 | 绘制ROI后点击"开始分析" | 请求体中roi_x/roi_y/roi_width/roi_height与Canvas绘制一致 |
| FE-ANALYSIS-008 | 未画ROI直接分析 | 不绘制ROI，直接点"开始分析" | 使用默认整帧ROI（x=0,y=0） |
| FE-ANALYSIS-009 | 进度条刷新 | 分析进行中观察进度条 | 进度百分比递增，与`GET /api/tasks/{id}`返回的progress一致 |
| FE-ANALYSIS-010 | 分析完成 | 等待分析结束 | 进度条100%，结果视频区域显示播放器，事件列表出现 |
| FE-ANALYSIS-011 | 分析失败 | 使用损坏视频或模型缺失 | 进度条变红，显示error_message |
| FE-ANALYSIS-012 | 结果视频播放 | 分析完成后点击播放 | 结果视频可正常播放、暂停、拖动进度 |
| FE-ANALYSIS-013 | 重复启动分析 | 任务已完成后再次点击"开始分析" | 提示"任务已完成，请重新上传视频"或同理提示 |

|---|---|---|---|

## 5. 历史事件页 `/history`

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|---|---|---|---|
| FE-HISTORY-001 | 事件列表分页 | 事件超过10条时 | 底部分页器可翻页，每页10条 |
| FE-HISTORY-002 | 时间范围筛选 | 选择开始和结束时间，点击查询 | 列表只显示该时间范围内的事件 |
| FE-HISTORY-003 | 置信度筛选 | 设置最低置信度0.5，点击查询 | 列表只显示confidence≥0.5的事件 |
| FE-HISTORY-004 | 组合筛选 | 同时设置时间+状态+置信度 | 三个条件同时生效 |
| FE-HISTORY-005 | 无匹配事件 | 设置不可能匹配的筛选条件 | 显示"暂无符合条件的事件" |
| FE-HISTORY-006 | 截图预览 | 点击某条事件的截图缩略图 | 弹出大图预览（Lightbox或对话框） |
| FE-HISTORY-007 | 视频回放 | 点击某条事件的"回放"按钮 | 播放对应结果视频 |
| FE-HISTORY-008 | 结果视频缺失 | 事件result_video_path为空 | "回放"按钮禁用（置灰），提示"结果视频生成中" |
| FE-HISTORY-009 | 筛选重置 | 设置筛选条件后点击"重置" | 所有筛选条件恢复默认，列表刷新为全部事件 |

## 7. 参数设置页 `/settings`

| 编号 | 测试项 | 操作步骤 | 预期结果 |
|---|---|---|---|
| FE-SET-001 | 读取默认参数 | 首次进入参数页 | 6个参数均显示默认值 |
| FE-SET-002 | 修改单个参数 | 修改detect_confidence为0.5，点保存 | 提示"参数已保存"，GET再次读取为0.5 |
| FE-SET-003 | 参数范围校验 | 输入detect_confidence=2.0，点保存 | 表单字段标红，提示"必须在0.1到1.0之间" |
| FE-SET-004 | 空值校验 | 清空某个参数，点保存 | 表单字段标红，提示"不能为空" |
| FE-SET-005 | 负值校验 | 输入min_vertical_distance=-10 | 表单字段标红，提示范围不合法 |
| FE-SET-006 | 恢复默认 | 修改多个参数后，点"恢复默认" | 所有参数恢复为系统默认值（0.35, 0.7, 80, 5, 0.7, 10） |
| FE-SET-007 | 重启后持久化 | 修改保存→重启后端→再次读取 | 参数保持修改后的值 |
