## 遇到的问题

1. Chrome全屏模式下Element Plus的el-dialog遮罩层未覆盖全屏——添加`:append-to-body="true"`
2. 演示机屏幕分辨率1366×768，HomeView流程图显示偏大——添加媒体查询`@media (max-width: 1366px)`

## 解决办法

- el-dialog全屏：`append-to-body="true"`确保遮罩层挂载到body
- 响应式：`@media (max-width: 1366px) { .flow-diagram { transform: scale(0.85) } }`
