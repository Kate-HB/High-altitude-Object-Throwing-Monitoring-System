## 完成结果

### 1. imgsz参数统一（960）

训练脚本 `train.py` 默认imgsz=960，但检测器未显式设置，存在不一致。

修改文件：
- `config/default.yaml`：新增 `imgsz: 960` 配置项
- `algorithm/detection/detector.py`：`__init__` 新增 `imgsz` 参数，`predict_frame` 和 `predict_video` 透传
- `algorithm/pipeline.py`：从 `settings` 读取 `imgsz`，异常兜底960

现在全链路imgsz统一为960：train.py → detector.py → pipeline.py → config/default.yaml

### 2. ONNX导出脚本

`scripts/export_onnx.py`：
- 基于ultralytics原生export
- 支持 opset/simplify/dynamic/half 四个导出选项
- CLI参数：`--model` `--output` `--imgsz` `--opset` `--no-simplify` `--dynamic` `--half`

### 3. 算法指标评估脚本

`scripts/evaluate.py`：
- 调用 `model.val()` 计算 Precision、Recall、mAP50、mAP50-95
- 输出推理速度（预处理/推理/后处理 ms）
- 结果自动保存到 `outputs/eval/`，含 plots

### 4. 演示视频录制脚本

`scripts/record_demo.py`：
- 调用 algorithm.pipeline 完整链路
- 三档阈值预设：default / sensitive / strict
- 自动检测视频尺寸，ROI默认覆盖画面底60%
- 输出结果视频 + 事件快照 + JSON摘要
