### P3 — 项目背景 (1min)
- 高空抛物社会危害（数据：每年XX起事故）
- 传统监控痛点：人工查看效率低、无法实时报警
- 行业现状：现有方案多为事后回溯，缺乏智能实时检测
- 项目目标：**实时检测→跟踪→行为判断→报警** 全自动闭环

### P4 — 系统架构总览 (1.5min)
- 架构图（前端→后端→算法三层）
  - 前端：Vue 3 + Element Plus（6页面SPA）
  - 后端：Python FastAPI + SQLite（RESTful API，Token认证）
  - 算法：YOLOv11 + IOU跟踪 + 行为判断（PyTorch + OpenCV）
- 数据流：上传视频→ROI选择→逐帧检测→IOU跟踪→行为分析→事件入库→报警展示
- 部署方案：Windows + RTX 3050 4GB + Conda环境

### P5 — 核心算法流程 (2min)
- YOLOv11目标检测：预训练模型微调，检出高空物体
- ROI区域过滤：用户自定义检测区域，减少误报
- IOU贪心匹配跟踪：帧间目标关联，构建运动轨迹
- 6参数行为判断：垂直位移、速度方向、加速度、轨迹线性度（最小二乘斜率法）、持续帧数、ROI通过比例
- 冷却时间机制：避免同一事件重复报警
- 结果可视化：检测框+轨迹线+ROI叠加，H.264编码输出

### P13 — 创新点 (1min)
1. **全自动闭环**：从上传到报警无需人工干预
2. **6参数综合行为判断**：结合运动学+统计学，降低误报
3. **ROI可配置**：适配不同楼栋/摄像头场景
4. **事件全生命周期管理**：检出→确认→误报→可追溯
5. **模块化架构**：算法接口合同解耦，可替换检测器/跟踪器

### P14 — 不足与展望 (1min)
- 已知不足：

  - 训练OOM问题（4GB显存限制，第4-6次训练OOM，需8GB机器）
  - 报警弹窗/声音未实现

- 展望方向：

  - DeepSORT替换IOU跟踪
