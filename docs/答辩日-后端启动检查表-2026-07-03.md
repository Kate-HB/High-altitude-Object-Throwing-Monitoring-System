## 常见问题速修

| 问题 | 原因 | 修复 |
|------|------|------|
| ModuleNotFoundError: onnxruntime | Conda环境缺少onnxruntime-gpu | `pip install onnxruntime-gpu==1.20.1` |
| Algorithm: "unhealthy" | ONNX模型未找到或加载失败 | 检查 models/best.onnx 是否存在 |
| Address already in use | 上次后端未正常关闭 | `taskkill /F /IM python.exe` 或换端口 |
| Database is locked | 上次写入未完成 | 重启后端自动恢复（WAL模式） |
